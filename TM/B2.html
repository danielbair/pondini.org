Time Machine - Troubleshooting

B2.  The Select Disk button doesn’t work

Previous        Frequently Asked Questions        Home     Troubleshooting      Contact      Next 

 

If the Select Disk button (Select Backup Disk or Change Disk on Leopard) in TM Preferences doesn't do anything, and you have a MobileMe subscription, try turning-off the Back To My Mac application temporarily, via System Preferences > MobileMe > Back to My Mac.
