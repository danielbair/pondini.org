Backup Structure

Time Machine Backups consist of a descending series of folders representing each Mac that was backed-up, each time it was backed-up, each volume (disk or partition), then within each volume, the entire file/folder structure of the original.

  1. •Local backups (on an internal or directly-connected external HD) are in a single Backups.backupdb folder.  If multiple Macs are backed-up to the same volume (by connecting each one in turn), they'll all be in the same Backups.backupdb folder, but in a separate sub-folder for each.

  2. •Network backups made over a network (Ethernet or WIFI) have the same structure internally, but the Backups.backupdb folder is in a separate sparse bundle disk image for each Mac (thus will have only one folder representing the computer.)
